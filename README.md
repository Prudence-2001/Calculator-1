# Simple Calculator Project

A simple, responsive calculator built using **HTML, CSS, and JavaScript**.  
It supports basic arithmetic operations, keyboard input, dark/light mode, and calculation history.

---

## Features

- Basic math operations: `+ - * /`
- Keyboard support (Enter, Backspace, Escape)
- Clear button (C)
- Green "=" button for calculations
- Red "C" button for clearing
- Dark / Light mode toggle
- Calculation history list
- Responsive design (works on mobile and desktop)

## How It Works

### 1. Input System
Users can click buttons or use the keyboard to enter numbers and operators.

### 2. Calculation
When "=" is pressed:
- The expression is evaluated using JavaScript
- The result is displayed in the screen
- The calculation is saved to history

### 3. History Feature
Every calculation is stored in a list so users can track previous results.

### 4. Theme Toggle
Users can switch between:
- Light mode
- Dark mode

## Keyboard Controls

| Key        | Action            |
|------------|------------------|
| Numbers    | Input numbers     |
| + - * /    | Operators         |
| Enter      | Calculate result  |
| Backspace  | Delete last char  |
| Escape     | Clear display     |
